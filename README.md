## Использование

В директории проекта вы можете запустить:

### `npm start`

Запустит приложение.\
Открыть [http://localhost:3000](http://localhost:3000) в вашем браузере.

### `npm run build`

Соберет сборку для продакшена.

### "deploy": "gh-pages -d build"

Деплой на github pages
